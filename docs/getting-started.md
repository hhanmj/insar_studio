# 快速开始

这份文档面向第一次使用 InSAR Studio 的用户，目标是帮你完成一次标准的数据准备流程。

## 1. 准备账号

首次使用建议先进入“设置”：

- Earthdata / ASF：用于 Sentinel-1 检索和下载。
- OpenTopography：用于 DEM 下载。
- 网络代理：如果你的网络无法直接访问数据源，可以在这里配置。
- 缓存目录：默认启用，用于保存任务状态、日志和必要缓存。

软件不会内置公共账号或密钥。每位用户都应使用自己的账号，并遵守对应数据源网站的使用规则。

## 2. 选择 AOI

AOI 可以来自：

- 地图手动绘制。
- 行政区边界。
- 本地边界文件。
- 手动经纬度范围。

正式项目建议上传权威边界文件；内置行政区更适合快速筛选和预览。

## 3. 检索 Sentinel-1 数据

进入资源下载后：

1. 选择 Sentinel-1 / ASF。
2. 设置日期、轨道方向、产品类型、极化等筛选条件。
3. 点击检索。
4. 在地图中查看影像范围，在列表中确认元数据。
5. 在 SAR 工作台勾选要下载的影像。

如果检索数量较多，优先选择覆盖率更高的数据可以减少无效下载。

## 4. 下载与恢复

下载任务进入下载中心后，可以查看：

- 当前速度和进度。
- 并发中的文件。
- 暂停、继续、结束、重试。
- 每个任务的详细日志。

软件会保存任务状态。重启后，暂停任务应继续停留在任务队列；结束、失败、超时任务进入历史记录，可按需恢复或重试。

## 5. 准备轨道和 DEM

- 精密轨道会根据 Sentinel-1 影像匹配 POEORB/EOF。
- DEM 可仅下载，也可下载后转换椭球高。
- DEM/GDAL 高级转换组件按需安装，不默认塞进主程序。

## 常见误区

- InSAR Studio 不做完整 InSAR 解算。
- 没有账号或 API Key 时，不应开始真实下载。
- 代理只影响需要联网的数据源，不改变本地文件处理。
- SARscape 对 DEM 命名和格式有特定要求，转换前请确认目标软件流程。
