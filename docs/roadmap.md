# 路线图

路线图用于说明方向，不代表所有功能都会在固定时间交付。

## 近期

- 完善下载中心工作台，让单景下载、暂停、继续和重试更稳定。
- 继续优化 AOI 多要素选择、绑定和地图高亮。
- 完善 DEM 下载、仅下载、下载并转换、本地转换的界面和日志。
- 优化安装版和更新检查体验。
- 清理公开仓库结构，补齐文档、模板和贡献入口。

## 中期

- 接入 Sentinel-2。
- 接入 Landsat。
- 接入 HLS。
- 学习成熟下载器的缓存判断、批量任务拆分、多图层和轻量安装逻辑。
- 完善组件化：DEM/GDAL、更多数据源插件和可选依赖按需安装。

## 长期

- 更完整的遥感数据准备工作台。
- 多数据源统一检索、筛选、下载和缓存。
- 更可靠的在线更新与安装包覆盖升级。
- 更适合教学、新手和项目批量管理的流程模板。
