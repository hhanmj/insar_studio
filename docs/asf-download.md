# Sentinel-1 / ASF 下载说明

InSAR Studio 当前重点支持 Sentinel-1 / ASF 场景，包括 SLC/GRD 检索、影像范围显示、元数据查看、下载队列和断点续传。

## 检索逻辑

检索通常由以下条件共同决定：

- AOI 几何范围。
- 日期范围。
- 产品类型：SLC、GRD 等。
- 轨道方向：升轨、降轨。
- Path / Frame。
- 极化：例如 VV、VH、VV+VH。

如果 AOI 是不规则边界，软件应尽量使用真实几何参与筛选；如果数据源接口只能使用外接矩形，界面需要提示用户，并在结果中优先展示覆盖率更高的影像。

## SAR 工作台

检索结果进入 SAR 工作台后，可以：

- 搜索日期、轨道、文件名等字段。
- 勾选单景或全选当前列表。
- 点击影像后在地图上高亮。
- 查看完整元数据。
- 对已检索或导入的影像启动下载。

“全选”表示选择当前可见列表：没有搜索条件时选择全部，有搜索条件时选择筛选后的列表。

## 下载队列

下载中心负责真实下载任务：

- 并发下载。
- 暂停和继续。
- 结束任务。
- 失败重试。
- 断点续传。
- 查看详细日志。

每个任务应绑定创建时的输出目录，避免用户后续修改目录后影响重试和断点续传。

## 网络与代理

如果用户开启代理，下载和检索应使用用户设置的代理；如果未开启代理，应使用系统默认网络。

常见失败原因：

- Earthdata / ASF 凭据未配置或已过期。
- 网络无法访问 ASF 或 Earthdata。
- 代理未正确工作。
- 目标目录无写入权限。
- 下载链接过期，需要重新获取。

## 手动导入

用户也可以通过 ASF 官方文件、URL、脚本或本地文件名导入候选影像。导入后应进入同一个 SAR 工作台，方便继续筛选、查看元数据和下载。
