# 支持与反馈

如果你在使用 InSAR Studio 时遇到问题，建议优先使用 GitHub issue。这样问题、修复过程和解决方案可以留给后续用户参考。

## 反馈入口

- 问题反馈：使用 Bug report 模板。
- 功能建议：使用 Feature request 模板。
- 使用讨论：可以在 issue 中说明你的处理场景、数据源和预期流程。
- 版本下载：查看 GitHub Releases。

## 提交问题前

请先确认：

- 是否使用最新版本。
- 是否已经配置 Earthdata/ASF 或 OpenTopography 凭据。
- 是否需要代理访问对应数据源。
- 下载中心日志里是否有明确错误码或失败原因。

请不要公开上传账号、Token、Cookie、完整签名下载链接、私有边界文件或未公开项目数据。
